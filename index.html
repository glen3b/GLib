GlenLibrary

A simple, general purpose C# library covering primarily XNA but also windows forms, convenience extension methods, and more.

Welcome to GLib

This library simplifies usage of various APIs available in the .NET framework, such as XNA, Windows Forms, and various other utilities including conversions to common types, such as integer. For example, to parse a string as an integer, you can just run: "Your string".ToInt() (extension method). If you find a bug, please report it on the issues page.

Building from source

To build this library from source, you need a C# compiler, preferably Visual Studio 2010, and, to build the XNA library, XNA game studio 4.0 or later. This project includes a Visual Studio 2010 solution file. If you want to use Visual Studio C# express, some projects will fail to load, but you can use these instructions.
