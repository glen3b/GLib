GlenLibrary

A simple general purpose C# library covering windows forms, XNA, extension methods, and more.

Download as .zip Download as .tar.gz View on GitHub

Welcome to GLib

This library simplifies usage of various APIs available in C#, such as XNA, Windows Forms, and conversions to common types, such as integer. For example, to parse a string as an integer, you can just run: "Your string".ToInt(). If you find a bug, please report it on the issues page.

Building from source

To build this library from source, you need a C# compiler at least (I recommend getting the IDE Visual C# express 2010) and, to build the XNA library, XNA game studio 4.0 or later. This project includes a Visual Studio 2010 solution file. If you cannot force-open this file in the express version, open the project files directly.
